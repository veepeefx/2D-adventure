## 2D Adventure Game (Unfinished)
This is a personal 2D game project that was started as an experiment but never finished. 
Development is stopped, and there are no plans to continue working on it.

#### Description
  - Custom made 2D level + textures (player sprites down below)
  - Basic player movement (running across the map)
  - Randomly generated trees scattered throughout the level

![Image](https://github.com/user-attachments/assets/3649cfab-d6d0-491e-be9a-95da4f06a564)

#### Future Addition Plans (not coming)
  - Items to gather
  - Inventory (with items)
  - Possibility to save/load progress
  - Multiple levels (mines, shopping area, forest etc.)
  - NPC's with interactions

Player Sprites are from: https://otterisk.itch.io/hana-caraka-base-character
